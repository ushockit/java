На основе моего примера, добавьте загрузка файла для продукта.
Добавьте модель "Комментарий" и свяжите его с моделью "Продукт" используя связь OneToMany.
Реализуйте соответствующий REST контроллер для добавление комментария к продукту.
Rest контроллер для комментирием должен иметь возможность
- добавление нового комментария к продукта
- получение всех комментариев к определенному продукту
- удаление комментария у определенного продукта
