Создайте консольный магазин продуктов.
Реализуйте возможности:
1. Просмотр продуктов заданной категории
2. Отображение всех продуктов (Название категории: Список продуктов)
3. Отображение продуктов в заданной ценовой категории
4. Отображение самого дешевого продукта
5. Отображение самого дорогого продукта
6. Отображение продуктов у которых цена выше среднего
7. Отображение продуктов у которых цена ниже среднего
8. Поиск продукта по названию


Задания должны быть реализованы используя Stream API
Функционал по добавлению/удалению/редактированию реализовывать не нужно, так что создайте предварительно 15-20 продуктов
